# UNLOX Investigation

Repository containing the Systems Investigation for the UNLOX ecosystem.

This repository preserves the investigation, its methodology, reasoning,
thinking-process artifacts and resulting product documentation.

## Repository Entry Point

Read the documents in the following order:

1. 02_ROADMAP.md
2. 01_STATUS.md
3. 00_MANIFEST.md

The roadmap is the operational control document for the investigation.
It determines current execution state and the next recommended work.

## Repository Principles

- One canonical Markdown artifact per investigation deliverable.
- Investigation before prescription.
- Documentation follows investigation.
- Changes are evaluated against the roadmap before execution.
- The repository is the canonical source of truth.

See `UNLOX_Documentation_Structure.md` for the repository constitution.
